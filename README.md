A high-fidelity prototype of WallEraser App. 

This prototype is made with Unity and should show main app feature: diminishing wall and get a look on the inside infrastructure of the Wall. 

The project was tested only as a 3D game with a keyboard as input. We do not guarantee that the game can be played with a VR headset. 

To execute start the game you will need: 
- UnityHub
- Android Studio and lts Android SDK
- Unity Version 2021.3.4f1 with Android Build Support installed
- Import all assets/dependencies
- Clone the master branch and open it with Unity
- The game can be started by pressing "Play sign" in opened project
